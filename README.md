# an-lisis-tendencias-youtube
### PROYECTO: ANÁLISIS DE TENDENCIAS DE YOUTUBE - STERLING & DRAPER ###

Enlace al Dashboard interactivo: https://public.tableau.com/views/AnlisisdeTendenciasdeYoutube-SterlingDraper/Dashboard1?:language=es-ES&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link


Descripción:  Este proyecto analiza las tendencias de video en YouTube utilizando Tableau. El dashboard incluye:

Historial de tendencias por volumen absoluto.

Distribución regional por categorías (Gráfico de pastel).

Análisis de cuota de mercado (100% acumulado).

Tabla de resaltado por categorías y regiones.

Instrucciones: Utilice los filtros ubicados en la parte superior del dashboard para segmentar los datos por "País" y "Rango de Fecha". El dashboard es totalmente interactivo.

Archivos incluidos: 
trending_by_time.csv (Datos originales).
Presentacion.pdf (Respuestas al equipo de marketing).
readme.txt (Este archivo).
